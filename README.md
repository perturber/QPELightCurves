## Installation Guide
0. **It is advisable to work in a conda environment:**
   ```bash
   conda create -n qpe_env jupyter matplotlib tqdm
   conda activate qpe_env
1. **Clone the repository:**
   ```bash
   git clone https://github.com/perturber/QPELightCurves.git
   cd QPELightCurves
2. **Install in editable mode:**
   ```bash
   pip install -e .
